# GPR_handoff主解析: RF基準、GPR前処理、カーネル比較、挙動診断

この文書は `GPR_handoff.zip` の主解析だけを対象とします。受領RFと同じ170例・固定10-foldを基準に、どの入力をどのfold内前処理へ通し、どのGPカーネルで予測したか、RFよりどこが改善し、どこで失敗するかを再現可能な形でまとめます。

最初に全体設計を短く確認する場合は、[データ → 特徴量 → 目的変数 → Held-out → モデル → 評価 → 結果 → 示唆](ANALYSIS_OVERVIEW_JA.md)を参照してください。特に固定10-foldとtrajectory group CVは未知にする対象が異なるため、同じ性能表へ無条件に混ぜません。

## 1. 先に要点

- `GPR_handoff` は170例の回帰データです。111個のbase特徴量と3,102個の `X_proc` 特徴量を使用します。
- 更新後の最良GPRは、角度を4つの分子軸座標へ整理し、`k_axis + k_environment + k_axis×k_environment + White` を使う相互作用GPです。固定10-fold OOFは `R²=0.973807` です。
- 受領RF報告値 `R²=0.908223` に対してR²は0.065584増え、RMSEは46.6%、MAEは49.3%減りました。従来首位Matérn 3/2 `R²=0.933874` に対しても改善しています。
- 軸・環境を分けない定常全体カーネルでは、Matérn 3/2、等方RBF、Rational Quadratic、Matérn 5/2が `R²=0.9328–0.9339` でほぼ同等です。
- 線形カーネルは `R²=0.662463` まで低下するため、非線形性は必要です。120次元RBF-ARDも `R²=0.872676` へ低下し、74.1%の長さ尺度が上限へ達しました。
- 固定foldは既知系列内補間です。暫定trajectoryをhold outすると相互作用GPも `R²=0.365`（同じprefix-group10）、モデル・gate選択込みの5×4 nested group CVは `R²=0.334` まで低下します。

## 2. `GPR_handoff` のデータ

### 2.1 観測単位と目的変数

- 標本数: 170
- 目的変数: `01_base_summary_first_angle.csv` の `y`。元PowerPointでは「ディラジカル性」と記載
- `y` の範囲: 1.89–52.24（平均43.489、標準偏差10.157）
- 識別子: `file_key`
- 評価fold: 10-fold、各fold 17例

同梱CSVにはディラジカル性の単位・算出法・電子状態計算条件がないため、本解析では数値回帰対象として扱います。値を百分率、エネルギー、占有数などの特定の物理単位へ読み替えることはしません。

3ファイルは `file_key` と行順が完全に一致している必要があります。読込時に、一意性、行対応、欠損、fold番号1–10を検査し、不一致があれば停止します。

### 2.2 入力CSV

| ファイル | 大きさ | 役割 |
| --- | ---: | --- |
| `01_base_summary_first_angle.csv` | 170 × 113 | `file_key`、`y`、111個のbase特徴量 |
| `02_Xproc_matched.csv` | 170 × 3,103 | `file_key`、3,102個のwideな `X_proc` 特徴量 |
| `03_cv_folds_seed123.csv` | 170 × 2 | `file_key`、固定fold番号 |

base特徴量には、距離しきい値 `d=3,5,7,10` ごとのMg/O数、逆距離の和・最大・上位3個、H3/H6間の差と和、最初に非ゼロとなる逆距離、原子種dummy、および方向特徴量が含まれます。

### 2.3 「角度」の意味

ここでの角度は化学結合角ではなく、共通座標系のxy平面上で見たC–Hベクトルの方位角です。

\[
\theta_3=\operatorname{atan2}(y_{H3}-y_{C3},x_{H3}-x_{C3}),\qquad
\theta_6=\operatorname{atan2}(y_{H6}-y_{C6},x_{H6}-x_{C6}).
\]

- `C3H3_angle_xy`: C3→H3の方位角
- `C6H6_angle_xy`: C6→H6の方位角
- `angle_diff_C3_C6`: 上記2方向の差を `[-pi, pi]` に丸めたもの
- `cos_angle_diff_C3_C6`, `sin_angle_diff_C3_C6`: 相対方向の周期表現
- `dot_C3H3_C6H6`, `dot_xy_C3H3_C6H6`: 2方向の平行性に関する内積

これらは座標系依存であり、全構造が同じ9-cell座標系で整列していることを前提に意味を持ちます。

## 3. `GPR_handoff` の前処理とモデル

### 3.1 情報リークを防ぐ単位

外側foldを1つテストとし、以下の学習を残り9 foldだけで行います。

1. 分散ゼロ列の選択
2. 平均0・分散1への標準化
3. `X_proc` のPCA
4. GPRのカーネルハイパーパラメータ推定

テストfoldには、訓練foldで決めた列選択、平均・標準偏差、PCA射影を適用するだけです。全170例でPCAや標準化を先に学習する処理は行いません。

### 3.2 角度の周期変換

周期版では、生の角度 `theta` を次の2変数へ置き換えます。

\[
\theta\longmapsto(\sin\theta,\cos\theta).
\]

これにより、数値上は遠い `-pi` と `+pi` が、方向として近いことを表現できます。現在の入力には角度差のsin/cosが既に含まれ、実装でも生の `angle_diff_C3_C6` からsin/cosを生成するため、角度差の情報が重複します。したがって、今後の厳密なablationでは重複を除き、同じカーネルで生角度と周期角度だけを比較する必要があります。

### 3.3 `X_proc` のPCA

3,102次元の `X_proc` について、訓練fold内で分散ゼロ列を除去し、標準化してからrandomized PCAを学習します。既定では上位8 scoreをbase特徴量へ結合します。PCA scoreは、成分ごとの分散差を残すため、結合前に再標準化していません。

これは受領RFの「PLSで残差を圧縮して第2のRFで補正する」二段階モデルとは異なります。現在のGPRは、base特徴量と教師なしPCA scoreを同時に入力する一段階GPRです。

### 3.4 GPRとカーネル

全候補でscikit-learnの `GaussianProcessRegressor` を使用し、目的変数を訓練fold内で正規化します。カーネルは共通して

\[
k(x,x')=\sigma_f^2 k_{\mathrm{space}}(x,x';\ell)
+\sigma_n^2\mathbf{1}[x=x']
\]

すなわち `ConstantKernel × spatial kernel + WhiteKernel` です。信号分散、長さ尺度、観測noiseは対数周辺尤度最大化、すなわち第二種最尤法で学習します。

公平なカーネル比較では、角度のsin/cos周期表現とfold内 `X_proc` PCA8を共通にし、空間カーネルだけを変更しました。

| 候補 | spatial kernel | 長さ尺度 |
| --- | --- | --- |
| `base_cyclic_xproc_pca8_matern12` | Matérn 1/2 = Exponential | 全次元共通 |
| `base_cyclic_xproc_pca8_matern32` | Matérn 3/2 | 全次元共通 |
| `base_cyclic_xproc_pca8_matern52` | Matérn 5/2 | 全次元共通 |
| `base_cyclic_xproc_pca8_rbf_iso` | RBF / Squared Exponential | 全次元共通 |
| `base_cyclic_xproc_pca8_rational_quadratic` | Rational Quadratic | 全次元共通、混合尺度alphaも最適化 |
| `base_cyclic_xproc_pca8_rbf_ard` | RBF / Squared Exponential | 特徴ごとのARD |
| `base_cyclic_xproc_pca8_linear` | Linear / DotProduct | 長さ尺度なし |

各関数形、isotropicとARDの違い、White noise、第二種最尤法は[GPRカーネルの関数形と比較結果](KERNELS_JA.md)に数式付きで整理しています。

## 4. 受領RFの再現方法

RFはRの `randomForest` と `pls::plsr(method="simpls")` を使用します。

1. 各外側foldの訓練データで、111個のbase特徴量から1,000本の回帰木RFを学習します。
2. 訓練データに対するRFのout-of-bag予測から残差を作ります。
3. `X_proc` の分散ゼロ列を訓練fold内で除き、標準化して、その残差を目的とするPLS 5成分を学習します。
4. PLS scoreから残差を予測する1,000本の第2RFを学習します。
5. テストfoldでは「base RF予測 + residual RF予測」を最終予測とします。

R 3.6以降の `sample()` 変更の影響を確認するため、`sample.kind="Rejection"` と旧 `"Rounding"` の両方を実行しましたが、今回の結果は同一でした。

## 5. `GPR_handoff` の評価と結果

`03_cv_folds_seed123.csv` の固定10-foldから全170例のout-of-fold予測を作り、次を計算します。

\[
R^2=1-\frac{\sum_i(y_i-\hat y_i)^2}{\sum_i(y_i-\bar y)^2},\quad
\mathrm{RMSE}=\sqrt{\frac1n\sum_i(y_i-\hat y_i)^2},\quad
\mathrm{MAE}=\frac1n\sum_i|y_i-\hat y_i|.
\]

GPRでは予測標準偏差、50/80/90/95%予測区間、coverage、区間幅、negative log predictive density（NLPD）も保存します。次表は、軸・環境を分ける前の全体定常カーネルscreenです。更新後の相互作用GPは6.8に示します。

| モデル | R² | RMSE | MAE | 95% coverage | NLPD |
| --- | ---: | ---: | ---: | ---: | ---: |
| GPR: Matérn 3/2 | **0.933874** | **2.604119** | 1.191774 | 0.952941 | 2.470919 |
| GPR: 等方RBF | 0.933505 | 2.611376 | 1.217731 | 0.935294 | **2.425369** |
| GPR: Rational Quadratic | 0.932962 | 2.622012 | 1.192167 | 0.941176 | 2.488398 |
| GPR: Matérn 5/2 | 0.932828 | 2.624628 | 1.232611 | 0.947059 | 2.470277 |
| GPR: Matérn 1/2 | 0.926884 | 2.738293 | **1.176260** | 0.952941 | 2.484521 |
| RF + residual PLS5（受領報告値） | 0.908223 | 3.067897 | 1.261954 | – | – |
| RF + residual PLS5（現環境再実行） | 0.900572 | 3.193211 | 1.339298 | – | – |
| GPR: RBF-ARD | 0.872676 | 3.613510 | 1.264649 | 0.876471 | 5.221269 |
| GPR: Linear | 0.662463 | 5.883489 | 3.698088 | 0.935294 | 3.173079 |

### 5.1 読み取れること

1. **方向・角度ブロックは有用と考えられます。** 元のRFの段階的比較では、summaryのみの `R2≈0.751` から角度ブロック追加後に `R2≈0.854` まで改善しています。ただし複数特徴量を同時に加えた比較なので、個々の角度の因果的寄与とは断定できません。
2. **全体定常カーネル内ではMatérn 3/2が首位ですが、上位は実質的に僅差です。** 等方RBFとの差は0.000369、上位4候補の幅も0.001046だけです。fold別RMSE首位回数は等方RBFが3回、Matérn 3/2とMatérn 1/2が各2回、Matérn 5/2とRational Quadraticが各1回で、特定カーネルが全foldを支配していません。
3. **RBF-ARDはこのデータでは不利です。** 120個の長さ尺度をfoldごとに推定し、全1,200個中889個が上限1,000へ達しました。170例に対して自由度が大きく、過剰パラメータ化になっています。
4. **点予測と確率予測の評価軸は一致しません。** R²はMatérn 3/2、NLPDは等方RBF、MAEはMatérn 1/2が最良です。不確実性の評価はR²だけでなくcoverage、区間幅、NLPDを併記すべきです。
5. **RFより有望ですが、改善は全試料で一様ではありません。** Matérn 3/2はRFより7/10 foldでRMSEが小さく、57.1%の試料で絶対誤差が小さくなりました。逆にfold 1ではRMSE 3.346対RF 1.130、fold 7では4.234対2.676でRFに負けています。
6. **予測区間の全体coverageだけでは十分ではありません。** 95% coverageは0.953ですが、予測標準偏差と絶対誤差のSpearman相関は0.207です。大誤差例を高不確実性として順位付けする能力は弱く、個別予測での不確実性解釈には注意が必要です。
7. **候補選択後の確定性能ではありません。** 同じ10-foldで候補を順位付けしているため、独立テスト、nested CV、または新規構造で事前固定した評価が必要です。

同梱された報告表のRF最終値 `R2=0.908223` と、同梱コードを現在の入力で再実行した値 `R2=0.900572` は一致しません。乱数サンプリング方式では説明できず、報告表作成時のコード、入力、または設定のいずれかが現在のZIPと異なった可能性があります。

## 6. 分子軸角度別の挙動と構造仮説

### 6.1 2本の方位角を一本の分子軸へ整理する

生の `C3H3_angle_xy` は−179.83°から178.15°にまたがるため、通常の数直線上でbin分けすると、同じ方向に近い−180°と+180°を別群にしてしまいます。また、C3→H3とC6→H6は3D内積が全例で−0.999960、xy方向差が176.99–179.98°で、ほぼ完全に反対向きでした。xy反平行からのずれは平均0.374°、最大3.012°にすぎません。

そこでC3→H3を180°反転し、C6→H6との円周平均を分子軸方位 `phi` としました。

\[
\theta_{3r}=\operatorname{wrap}(\theta_3+\pi),\qquad
\phi=\operatorname{atan2}(\sin\theta_6+\sin\theta_{3r},
                           \cos\theta_6+\cos\theta_{3r}),
\]

\[
\delta=\left|\pi-\left|\operatorname{wrap}(\theta_3-\theta_6)\right|\right|
\]

ここで `phi` は共通xy座標系における一本の分子軸方位、`delta` は完全な反平行からのずれです。`phi` の範囲は−22.95–60.94°でした。したがって現在の角度ブロックは、実質的に「一本の軸の向き + 非常に小さな歪み」であり、3つの独立した広い角度自由度ではありません。

3D内積とxy内積には

\[
\mathrm{dot}_{3D}-\mathrm{dot}_{xy}=u_{3z}u_{6z}
\]

の関係があります。2本がほぼ完全に反平行なので `u6z≈−u3z` とみなし、`sqrt(−u3z u6z)` を分子軸のz成分絶対値、`asin` を面外傾き絶対値のproxyとして併用します。これは元座標から各ベクトルのz成分を直接再計算した値ではなく、反平行性を使った近似です。

### 6.2 角度帯ごとの目的変数と予測性能

同じ固定10-fold OOF予測を `phi` の20°幅で事後層別しました。帯内では目的変数の分散が大きく異なるため、帯内R²よりRMSEとMAEを主に読みます。

| `phi` | n | y 平均 | y SD | 最良GPR | GPR RMSE | Matérn 3/2 RMSE | RF RMSE | 全体首位GPRのRF比 |
| --- | ---: | ---: | ---: | --- | ---: | ---: | ---: | ---: |
| −30–−10° | 24 | 46.098 | 7.648 | Matérn 5/2 | 2.956 | 3.086 | 4.338 | 28.9%改善 |
| −10–10° | 17 | 45.794 | 4.381 | RBF-ARD | 1.068 | 1.542 | 2.132 | 27.7%改善 |
| 10–30° | 36 | 47.940 | 2.721 | Matérn 3/2 | 1.026 | 1.026 | 1.934 | 46.9%改善 |
| 30–50° | 52 | 42.576 | 8.958 | Matérn 3/2 | 1.758 | 1.758 | 3.585 | 51.0%改善 |
| 50–70° | 41 | 38.256 | 15.246 | Matérn 1/2 | 3.525 | 4.088 | **3.129** | 30.6%悪化 |

ここから次を読み取れます。

1. **Matérn 3/2の全体首位は、広い中間角度帯での強さによります。** 10–50°の88例でRMSEが1.026–1.758と低く、この帯では連続で適度に滑らかな関数という仮定がよく合います。
2. **50–70°は別の挙動です。** `y` の標準偏差が15.246と最大で、41例中9例が `y<30` の低応答枝へ入ります。この帯だけでMatérn 3/2の全SSEの59.4%を占め、95% coverageも0.829へ低下します。
3. **粗い関数またはしきい値分割が高角度帯に合います。** 同帯では連続だが非微分なMatérn 1/2がGPR内首位で、全体首位はRFです。滑らかなMatérn 3/2・5/2・RBFは、特徴空間で近い高応答枝と低応答枝の間を平均化しやすいと考えられます。
4. **−10–10°のRBF-ARD首位は仮説生成に留めます。** n=17の事後比較であり、RBF-ARDは全体では `R²=0.872676` と低いので、この帯だけにARDを採用する根拠にはまだなりません。
5. **Matérn 1/2のMAEが全体最良で、Matérn 3/2のR²/RMSEが最良なのは矛盾しません。** 粗いMatérn 1/2は一部の枝分かれに強く絶対誤差を抑える一方、中間角度の多数例ではMatérn 3/2の二乗誤差がより小さいためです。

`delta` も3群で集計しましたが、全範囲が0–3.012°しかありません。これは平行・直交などの異なる配向レジームではなく、他の局所構造と相関する微小なずれとして扱うべきです。

### 6.3 「角度データが足りない」ことが原因か

各テスト例について、同じfoldを除いた訓練例までの最小分子軸角距離と、5°以内の訓練例数を計算しました。

- 最小角距離とMatérn 3/2絶対誤差: Spearman `rho=-0.009`
- 5°以内の訓練例数と絶対誤差: Spearman `rho=0.152`

高角度帯自体にも41例あります。少なくとも一次元の角度被覆だけを見る限り、大誤差は「近い角度の訓練例がない」ためではありません。同じ角度でも別のMg/O局所環境により応答が分かれる、条件付きの異質性・相互作用が主な候補です。

### 6.4 Mg/O局所環境との対応

元PowerPointは、従来RFの特徴重要度から「H6側ではMgが最初に現れる距離または距離内個数、H3側では近接Mg/Oの近さの強さがディラジカル性に効く」と考察しています。また残差解析では「C3H3/C6H6ベクトル方向とz≈0が重要に見えるが、zだけかxy平面内方向も関係するかは未確定」としています。以下は、この資料上の仮説を現在のOOF予測と要約CSVで定量化した診断です。

角度から直接作られた7列を除き、base特徴量と `phi` の単変量Spearman相関を調べると、`first_invd_LH6_atomO` が `rho=-0.826` で最も強く対応しました。分子軸30°未満の77/77例ではH6側で最初に現れる原子種dummyがO、30–50°では4/52例、50–70°では2/41例だけがOでした。`Other` dummyはこれらの群で0なので、dummyの基準水準を踏まえると大半がMg側へ切り替わっています。これは、分子軸方位が結晶格子のどの原子種へ向くかを反映し、約30°付近にカテゴリ的な幾何変化があることを示唆します。

さらに50–70°の中で、低応答9例 (`y<30`) と残り32例を比較すると次の差がありました。

| H3近傍特徴 | 低応答9例 | 他32例 | 全170例SDで割った平均差 |
| --- | ---: | ---: | ---: |
| `O_H3_count_d5` | 7.333 | 0.969 | +2.305 SD |
| `Mg_H3_count_d5` | 6.667 | 0.906 | +1.983 SD |
| `sum_invd_LH3_d5` | 3.492 | 0.422 | +2.151 SD |
| `first_O_H3_d5` | 3.033 | 5.303 | −1.917 SD |
| `first_Mg_H3_d5` | 3.139 | 5.388 | −1.877 SD |

特徴名の `d5` は受領データの距離しきい値5を表します。低応答枝ではH3近傍のMg/O数と逆距離和が大きく、最初のMg/O距離が短いため、「高角度で、H3方向がMg/Oの密な近接環境へ入ると低応答側へ移る」という構造仮説が得られます。H6側の距離・総数にも同方向の差がありますが、上位差はH3側に集中しました。

面外傾きproxyも同時に異なります。

| 50–70°内の群 | n | 面外傾きproxy平均 | 中央値 | 範囲 |
| --- | ---: | ---: | ---: | ---: |
| `y<30` | 9 | 3.15° | 0.85° | 0.24–9.12° |
| `y>=30` | 32 | 19.80° | 10.07° | 0.00–80.29° |

低ディラジカル性9例はすべてほぼ面内ですが、面外傾き10°未満の高応答例も16例あります。したがって「z≈0だけ」が十分条件ではありません。現在の最も具体的な仮説は、(a) 分子軸がxyで50–70°方向、(b) C–H軸がほぼ面内、(c) H3近傍にMg/Oが密で近い、という組み合わせでディラジカル性が低下する、というものです。これは、角度・dot・距離しきい値・カテゴリを分割できるRFがこの帯で強いこととも整合します。

これは因果結論ではありません。`y<30` の閾値、角度帯、比較特徴は結果を見た後に選び、Mg/Oカウント・逆距離・最短距離は互いに強く相関します。また、現在の3 CSVは要約特徴であり、元3D原子配置を直接可視化していません。現段階で言えるのは「低応答枝と局所配位記述子が強く共変する」までです。

### 6.5 構造系列を丸ごと未知にした補助評価

高角度・低応答9例の帰属を調べると、ファイル名prefix `0-0-2-16` に3例、`0-0-3-18` に6例が集中していました。`0-0-3-18` の7例では最後の数値tokenと `y` のSpearman相関が−1.0で、suffix 7から14へ進むにつれ `y=39.07` から1.89へ連続的に低下します。ファイル名prefixが同一構造系列または反応座標を表す可能性があります。

一方、指定固定foldは `0-0-2-16` を3 fold、`0-0-3-18` を5 foldへ分け、30 prefixの多くを複数foldへ分散しています。この分割でのRF/GPR比較は依頼どおり公平ですが、同じ系列の近傍構造を使う「系列内補間」の評価です。未知の分子構造系列への外挿性能とは区別する必要があります。

そこで、最後のhyphen tokenを除いたprefixを候補series IDとし、30系列を分割せず3系列ずつhold outする10-fold GroupKFoldを補助的に実行しました。base標準化、`X_proc` PCA8、GPRハイパーパラメータ推定は各group訓練fold内だけで行っています。

| --- | ---: | ---: | ---: | ---: | ---: |
| Rational Quadratic | **0.204954** | **9.029633** | **4.472770** | 0.900000 | 4.551189 |
| Matérn 1/2 | 0.144438 | 9.366987 | 4.726188 | **0.905882** | **4.327622** |
| Matérn 3/2 | −0.053689 | 10.395140 | 5.276053 | 0.829412 | 4.569863 |
| Matérn 5/2 | −0.128628 | 10.758446 | 5.512620 | 0.823529 | 4.679804 |
| RBF | −0.227962 | 11.221902 | 6.059563 | 0.835294 | 4.745282 |
| Linear | −1.140907 | 14.817432 | 9.188284 | 0.735294 | 6.644828 |

この結果は主たる固定fold結果を取り消すものではなく、問いが違います。

- 固定10-fold `R²=0.933874`: 既知の構造系列に属する未観測点を補間する用途
- prefix-group 10-fold 最良 `R²=0.204954`: 候補系列そのものが未知の用途

この全体定常カーネルscreenでは、多尺度のRational Quadraticと粗いMatérn 1/2が滑らかなMatérn 3/2・5/2・RBFより相対的に良く、非定常性・急なregime変化を示唆します。6.8で相互作用GPを同じsplitへ追加するとR²は0.365へ改善しますが、構造外挿はなお未解決です。受領二段R RFも同じgroup5/group10で再fitし、`gpr_handoff_trajectory_model_comparison.csv` で直接比較します。

この補助評価はprefixの意味を推定したものです。命名規則を確認し、prefixが本当に同一反応経路、初期構造、分子familyなどの独立単位を表す場合にのみ、正式なgroup splitとして採用します。別のtokenが実験上の独立単位なら、その定義で再分割します。

### 6.6 次に有望なモデルと検証順序

現在の170例で次を同じ外側10-foldに組み込み、内側foldだけで選択します。

1. **評価単位の確定:** file_key各tokenの物理的意味を確認し、実運用で未知となる構造系列を丸ごと外側foldへ出します。内側CVでも系列を分け、同系列近傍からのリークを防ぎます。
2. **物理的に整理した角度表現:** 重複した生角度・角度差を、`sin(phi)`, `cos(phi)`, `delta` へ置換します。まず同じMatérn 3/2でablationし、精度を維持したまま冗長性を減らせるか確認します。
3. **角度×局所環境の明示的相互作用:** H3/H6のMg/O数、最初の距離、逆距離和、原子種dummyを少数群へ絞り、`phi` との積・差、または `k_env + k_axis + k_env*k_axis + White` の加法・積カーネルを比較します。積項が「配向によって環境効果が変わる」仮説に対応します。
4. **regime別mixture-of-experts / treed GP:** 予測時に利用できる `phi` と局所環境だけでgateを学び、中間角度にはMatérn 3/2、高角度の枝分かれ領域にはMatérn 1/2またはRFを使います。50°や `y<30` を全データで固定するとリークするため、境界と専門モデルは内側CVで決めます。
5. **非定常・change-point GP:** `phi` に沿って滑らかさや長さ尺度が変わるカーネルを使います。小標本ではmixture-of-expertsより不安定になり得るため、まず上の単純な相互作用モデルを優先します。
6. **階層・multi-task GP:** 実験条件・分子種・構造familyの正しいラベルが別途得られる場合に有望です。現在のhandoff 3 CSVには明示的なtask定義がないため、ラベルなしで導入する優先度は低いです。
7. **元構造へ戻れる場合の回転不変特徴:** 各C–H軸の極角・方位角を直接再計算し、分子軸に沿ったMg/O位置の射影、軸からの垂直距離、H3/H6側の符号付き非対称性を作ります。絶対方位そのものより「分子軸に対して原子がどこにあるか」を表すため、xy方位とz傾きを分離でき、物理解釈と別座標系への一般化が改善します。

最優先は (1) の評価単位確認です。そのうえで (2) の整理表現と (3) の少数の相互作用を試します。現在のデータだけで実装でき、仮説を直接検証でき、120次元RBF-ARDのような過剰パラメータ化を避けられます。その次にnested group CV付きの (4) を比較します。

### 6.7 上記候補を実装した結果

6.6のうち、file_key診断、4変数角度表現、加法・積カーネル、mixture-of-experts、nested group CVを実装しました。元3D座標だけはhandoffに含まれないため、座標が届いた時点で実行できる特徴生成器までを追加しています。

生角度・差・内積の7列は次の4変数へ置換しました。


\[
(\sin\phi,\cos\phi, |\theta_z|,\delta),
\]


- `phi`: 反転C3→H3とC6→H6を円周平均した分子軸xy方位
- `|theta_z|`: 2内積の差と反平行性から求めた面外傾き絶対値proxy
- `delta`: 完全な反平行からのずれ

同じ全体Matérn 3/2へ入れた固定10-fold結果は `R²=0.933092` で、従来の重複角度表現 `0.933874` とほぼ同じでした。したがって、情報をほとんど失わず、角度の意味と次元を整理できています。

### 6.8 軸×環境の相互作用GP

このカーネルは、初めから候補を増やす目的で導入したものではありません。全体Matérn 3/2の誤差が50–70°へ集中した一方、この帯には十分な標本があり、最寄り訓練角と誤差も無相関でした。低応答枝だけH3近傍Mg/Oが密・近距離で、面外傾きが小さかったため、「角度の主効果」や「環境の主効果」ではなく、**分子軸の配置が局所環境効果を変える**という仮説が残りました。

全体定常カーネルは全特徴を一つの距離へまとめ、加法カーネルは軸と環境の効果を独立に足します。積カーネルは、二試料の軸類似度と環境類似度を掛けるため、両方が似るときだけ強く情報を共有します。functional ANOVA / tensor-product GPとして自然な最小拡張であり、結果を見て50°にhard boundaryを置くより連続的です。Matérn 3/2は従来screenの全体首位だったため、軸・環境それぞれの最初の関数族に採用しました。

入力を4個の軸特徴 `a` と、残りのbase + `X_proc` PCA8である環境特徴 `e` に分け、次をfitしました。

元baseは `file_key`,`y` を除く111列です。そこから方向の重複7列を4軸特徴へ置換し、残る104列のうち各訓練foldで分散ゼロの2列を除いた102列を環境baseとします。環境 `e` はこの102列とfold内PCA8の合計110次元、設計行列全体は軸4＋環境110＝114次元です。4軸列の正式名、除外した7列、分散ゼロ2列は[カーネル文書 7.2](KERNELS_JA.md#72-関数形)に列挙しています。


\[
k((a,e),(a',e'))=
k_{axis}(a,a')+k_{env}(e,e')+
k_{axis}(a,a')k_{env}(e,e')+k_{white}.
\]


各軸・環境カーネルはMatérn 3/2です。3個の信号分散、加法2個＋積項内2個の計4個の長さ尺度、White noiseを各訓練foldの第二種最尤法で推定します。積項は「同じ環境差でも分子軸が違うと効果が変わる」、またはその逆を表します。

受領RFと同じ固定10-foldの結果は次のとおりです。

| モデル | R² | RMSE | MAE | 95% coverage | NLPD |
| --- | ---: | ---: | ---: | ---: | ---: |
| **軸×環境 相互作用GP** | **0.973807** | **1.638955** | **0.639434** | 0.976471 | 2.389596 |
| 軸+環境 加法GP | 0.955026 | 2.147600 | 1.057447 | 0.964706 | **2.229022** |
| compact角度 全体Matérn 3/2 | 0.933092 | 2.619467 | 1.194784 | 0.958824 | 2.468694 |
| 従来角度 全体Matérn 3/2 | 0.933874 | 2.604119 | 1.191774 | 0.952941 | 2.470919 |
| 受領RF報告値 | 0.908223 | 3.067897 | 1.261954 | — | — |

相互作用GPは従来Matérn 3/2よりR²が0.039933高く、RMSEを37.1%、MAEを46.3%減らしました。受領RF報告値に対してはR²が0.065584高く、RMSEを46.6%、MAEを49.3%減らします。PCA乱数seedを123、4,123、9,123に変えた感度分析でも `R²=0.973755–0.973798` で、数値は安定していました。

固定foldの角度帯RMSEも全帯で改善しました。

| 分子軸方位 | compact全体M3/2 | 加法GP | 相互作用GP | RF |
| --- | ---: | ---: | ---: | ---: |
| −30–−10° | 3.111 | 2.543 | **1.511** | 4.338 |
| −10–10° | 1.557 | 1.359 | **0.828** | 2.132 |
| 10–30° | 1.039 | 1.217 | **0.478** | 1.934 |

高角度帯でも相互作用GPがRFを上回ったため、以前の「高角度はRFが最良」という結論は、全体等方カーネルまでの比較に限定されます。角度と環境の積を明示すると、高角度枝の一部を連続モデルでも表せます。

同じprefix-group10で比較すると、相互作用GPは `R²=0.365228, RMSE=8.068315` で、従来首位Rational Quadratic `R²=0.204954, RMSE=9.029633` より改善しました。ただし固定foldの `R²=0.974` からは大きく低下します。相互作用により系列外挿は改善したものの、解決したとは言えません。

group10各foldの最適化では、軸加法分散は10/10 foldで下限0.01、環境加法分散も9/10 foldで下限0.01でした。一方、積項分散は0.956–2.047（中央値1.64）でした。少なくともこのカーネル分解では、改善の中心が加法項ではなく軸×環境項であることを支持します。ただし、成分分散は相関した特徴下での周辺尤度最適化値であり、物理的寄与率や因果効果ではありません。

### 6.9 mixture-of-expertsとnested group CV

`phi < t` をcompact角度Matérn 3/2、`phi >= t` をMatérn 1/2またはPython RFとするhard gateを作り、`t=40°,45°,50°` を候補にしました。RF専門家は、受領Rのresidual PLS5 RFではなく、同じcompact base + fold内PCA8へfitするPython RFです。

暫定trajectory group 30系列を外側5-fold、内側4-foldとし、12候補のモデルと境界を内側RMSEだけで選びました。外側foldの勝者はRF gate 40°が3回、RF gate 50°が1回、相互作用GPが1回でした。strict nested OOFは次のとおりです。

| 評価 | R² | RMSE | MAE |
| --- | ---: | ---: | ---: |
| 内側選択を含むstrict nested | 0.334491 | 8.261349 | 4.535787 |
| 相互作用GPを固定した同じ外側5-fold | 0.465545 | 7.403375 | 4.513588 |
| M3/2→M1/2、40°gateを固定 | 0.409067 | 7.784720 | 4.698969 |

内側ではRF gateが多く選ばれたのに、外側で選択損失が出ています。170例・30系列ではgateと専門家の同時選択がまだ不安定です。mixtureは高角度帯だけなら有用で、prefix-group10の50–70° RMSEはRF gate 40°が9.452、Matérn 1/2 gate 40°が10.706、相互作用GPが12.329でした。しかし全角度を合算すると相互作用GPが首位です。現段階では相互作用GPを主候補、mixtureを高角度枝の診断・将来候補とします。

### 6.10 file_keyと外側group

### 6.13 受領二段R RFのtrajectory Held-out

`rf_reference.R` のモデル構造・乱数設定・木数を変えず、fold表だけを暫定trajectory groupへ置き換えました。したがって、これは高角度だけPython RFにしたmixtureではなく、受領モデルそのものの未知trajectory評価です。

| split | base RF R² / RMSE / MAE | base＋残差PLS5/RF R² / RMSE / MAE | Product GP R² / RMSE / MAE |
| --- | ---: | ---: | ---: |
| trajectory group5 | 0.073894 / 9.745507 / 5.203721 | 0.025996 / 9.994348 / 5.451921 | **0.465545 / 7.403375 / 4.513588** |
| trajectory group10 | −0.046150 / 10.357884 / 5.304266 | −0.212622 / 11.151589 / 5.720765 | **0.365228 / 8.068315 / 4.852925** |

固定10-foldでは残差補正がbase RFを `R²=0.853187` から0.900572へ改善しました。しかしtrajectory Held-outではgroup5、group10の両方で補正後が悪化しました。`X_proc` PLS5による残差補正は同じtrajectory内の近傍点には有効でも、系列を跨いで移らないパターンを学んでいる可能性があります。したがって補正段の採否・PLS成分数も、将来は外側trajectoryを保ったnested group CV内で選ぶ必要があります。

角度帯別RMSEでは、Product GPがgroup10の5帯すべて、group5の4帯で二段RFを上回りました。group5の−30–−10°のみRF 6.318、GP 6.855ですが、同帯MAEはGP 2.581、RF 4.181です。50–70°はProduct GPでもgroup5 11.561、group10 12.340で最大であり、未知trajectory高角度枝が最も難しいという構造的な結論は変わりません。

## 主解析で次に行うべき検証

1. 構造生成者に `file_key` token 1–5の正式定義を確認し、実運用上の独立単位に対応する外側groupを固定する。
2. 170個の `file_key` に対応する元3D座標を入手し、分子軸へのMg/O射影、垂直距離、H3/H6非対称性を追加する。
3. Product GP、受領base RF、残差補正の有無、PCA/PLS成分数を、正式なnested group CVの内側だけで選ぶ。
4. 正式な実験条件・構造familyラベルが得られた場合、groupをtaskとする階層／multi-task GPをProduct GPと比較する。
5. 高角度trajectoryを追加してから、Matérn 3/2→Matérn 1/2/RF mixtureの境界を内側CVで再評価する。
6. 予測区間はcoverage curveと必要に応じたgroup-conformal calibrationで外挿時の較正を確認する。
7. 最終モデルを事前固定し、完全に新しいtrajectoryを一度だけ独立評価する。

主解析の実装は `src/chemistory_gpr/kernels.py`、`handoff.py`、`nested_group.py`、`scripts/run_rf_trajectory.py` にあります。固定fold、trajectory Held-out、角度帯、相互作用成分の結果は `results/gpr_handoff_*.csv` に保存しています。
